# Hits-Assignment

<h2>Hits Frontend 과제</h2>

### 기능
**RESULT**
- 여러 유저들의 Foxtrot 과 Golf를 볼 수 있습니다.
- 각 유저별 세부 Foxtrot 과 Golf를 볼 수 있습니다.
- 각 유저별로 세부내역을 선택해서 Result 페이지에서 묶음 처리 할 수 있습니니다.
- 데이터별로 유저데이터를 오름차순 내림차순으로 졍렬 할 수 있습니다.
